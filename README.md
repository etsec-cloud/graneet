# Exercice Graneet

Bonjour, 
Voici mon test pour graneet, le front est en react, le back en nest et la bbd est un mongodb atlas

## Cloner le git
> git clone https://github.com/etsec-cloud/graneet.git

## Acceder au front
> cd app 

> npm i 

>  npm run start

## Lancer le back

> cd api

> npm i 

> npm run start

Pour se connecter Ã  la BDD les variables d'environnement sont dans le mail et sont a mettre dans un fichier .env dans graneet/api
